- Sort colors => left, mid, right algorithm
- Kaydane algorithm => add , update and if condition overflows then reset
- Print subarray => two start variables
- Increment by 2 for some iterators

- Backtrack algorithm for generating all permutations
- Trick to find next permutation

- Traverse from end in some cases
  - Add only maximum element from end [leaders]

- Pattern:
  - let something = true;
  - for() if() something = false

- Longest consecutive sequence [Hashset algorithm]
  - Find starting point [Point that have no prev]
  - while next element is present, len++

- Set Matrix Zero
  - [Brute] Make the whole row zero and whole col zero
  - [Optimal] Mark the first row element and first col element to zero
    - Except Matrix[0][0]

- Transpose of a Matrix
  - i : 0 => n
  - j : i+1 => n
  - [i][j] <=> [j][i]

- Reverse list using Stack

- Four pointers [Spiral matrix]

- Subarray sum with negative values
  - PrefixSum Map [Like a Kaydane algorithm]
